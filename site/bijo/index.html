Author Proposal (BIJO)

BIJO is WayChain's proposal markup. Write the what + why; the pipeline hashes it and opens a governance proposal. Authoring surface lives in the Binary Journal dApp (#109) — this is the forward/vote/task layer.

Track This Proposal

Truth-first: the proposal is created on the live governance precompile (0x1D); voting is Dox_Dev-weighted (Direct/Quadratic/Futarchy) per chain rules. On approval, the dev task is created via TaskRegistry (0x23) — the same autopilot/dev-task path the Verify Ladder (#111) consumes. No marketing-tier numbers are hardcoded as on-chain truth.
