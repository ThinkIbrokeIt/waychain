Getting Started

WayChain is live at chain ID 10008 (0x2718). Blocks every 1 second. Instant finality. One verified human = one voice.

This guide gets you from zero to querying the chain in under 5 minutes.

1. Check the Chain Is Alive

No wallet needed. No tokens needed. Any HTTP client works:

# Get chain ID
curl -s -X POST https://api.waychain.org \
  -H 'Content-Type: application/json' \
  -d '{"jsonrpc":"2.0","method":"eth_chainId","id":1}'

Expected response: {"jsonrpc":"2.0","result":"0x2718","id":1}

# Get latest block number
curl -s -X POST https://api.waychain.org \
  -H 'Content-Type: application/json' \
  -d '{"jsonrpc":"2.0","method":"eth_blockNumber","id":1}'

Expected: an increasing hex number. WayChain produces one block per second.

# Get block by number (replace with actual hex)
curl -s -X POST https://api.waychain.org \
  -H 'Content-Type: application/json' \
  -d '{"jsonrpc":"2.0","method":"eth_getBlockByNumber","params":["0xC0000",false],"id":1}'

3. What You Need to Deploy

WayChain requires Dox_Dev Level 2+ to deploy smart contracts. This is enforced at the protocol level — the EVM itself rejects deploys from unverified addresses.

Why? No anonymous rug-pulls.

Every deployer is a verified human. Bad actors cannot re-verify after revocation. This is not KYC — your identity is encrypted, disclosed only on verified legal request with on-chain approval via community vote.

For now, badge applications are handled manually. Open an issue on the monorepo with your WayChain address to begin verification.

4. Developer Resources

WayChain uses SHA256, not keccak256

All function selectors, storage keys, and transaction hashes use SHA256. Standard EVM tooling (ethers.js, Foundry) uses keccak256 — adapt your ABI selectors accordingly.

Key differences from standard EVM:

  • Binary transaction format (not RLP) — see consensus/serialize.go
  • Ed25519 signatures (not ECDSA)
  • SHA256-based selectors (not keccak256)
  • 20 native precompiles at addresses 0x0C-0x20
  • Fixed fees: 0.001/0.005/0.01 WAY (not auctioned)
  • 1-second block time with instant finality
