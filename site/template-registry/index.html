Template Registry

Compile · Register · Deploy

Write Solidity, compile in-browser, register a reusable template, and deploy instances on-chain via the TemplateRegistry precompile (0x26).

1 · Write Solidity

2 · Register Template (0x26)

Requires registrar role OR Dox_Dev L2+. Selector registerTemplate 0x7cbd749e.

3 · Deploy From Template (0x26)

Requires Dox_Dev level per template type. Selector deployFromTemplate 0x1de26edf. Contract address = SHA256(caller+blockNum)[:20].

Lookup

Truth-first: compilation runs locally via solc-wasm (loaded from CDN). Registration/deploy broadcast via TemplateRegistry (0x26). The dApp's own Solidity app-layer uses the Keccak256 precompile (0x21); the precompile itself uses sha256 selectors (two layers, both correct).
